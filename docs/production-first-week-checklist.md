# Checklist Primera Semana KUPAN

## Revisión diaria

- Verificar login admin.
- Revisar reservas del día.
- Revisar tokens bajos.
- Revisar planes vencidos.
- Revisar WOD.
- Revisar errores reportados por alumnos.
- Revisar Coach mode antes de clases.
- Confirmar que no hay reservas duplicadas.
- Confirmar que no hay dobles descuentos ni dobles devoluciones.

## Señales para detener apertura

- Fallo de login recurrente.
- Tokens incorrectos.
- Reserva duplicada.
- Doble devolución.
- Alumno viendo datos ajenos.
- Admin o Coach inutilizable.
