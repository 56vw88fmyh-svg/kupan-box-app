# Guía Rápida KUPAN

## Alumno

1. Entra a la app.
2. Inicia sesión.
3. Revisa tu plan y tokens.
4. Entra a Reservas.
5. Elige día y clase.
6. Toca Reservar.
7. Revisa tus reservas en Perfil.

## Coach

1. Inicia sesión.
2. Entra a Modo Coach.
3. Revisa la clase actual.
4. Marca asistencia o no show solo cuando corresponda.

## Admin

1. Inicia sesión.
2. Entra a Admin.
3. Presiona Actualizar datos.
4. Revisa alumnos, membresías, reservas y WOD.
