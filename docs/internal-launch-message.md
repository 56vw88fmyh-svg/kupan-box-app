# Mensaje Lanzamiento Interno KUPAN

Equipo KUPAN:

Vamos a iniciar una prueba interna supervisada de la app KUPAN.

Durante esta etapa la usaran pocas personas del box para revisar:

- reservas;
- cancelaciones;
- tokens;
- WOD;
- perfil;
- modo coach;
- panel admin.

Importante:

- Si ves algo raro, no repitas la accion.
- Avisa por el canal interno definido.
- Adjunta captura, hora y que estabas intentando hacer.
- Las operaciones sensibles seran revisadas diariamente.

Esta no es una apertura masiva para todos los alumnos. Es una prueba controlada para asegurar que la app funcione bien antes de ampliarla.
