# Instalación PWA KUPAN

## iPhone

1. Abre la URL en Safari.
2. Toca Compartir.
3. Elige "Agregar a pantalla de inicio".
4. Abre KUPAN desde el ícono.

## Android

1. Abre la URL en Chrome.
2. Toca el menú.
3. Elige "Instalar app" o "Agregar a pantalla principal".
4. Abre KUPAN desde el ícono.

## Actualización

Si aparece "Nueva versión disponible", toca "Actualizar app".
